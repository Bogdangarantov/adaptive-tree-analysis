# 🎓 Дипломна робота
## Веб-платформа для візуалізації та експериментального аналізу самобалансувальних дерев

## Що це за проєкт

`adaptive-tree-analysis` — це платформа для дослідження самобалансувальних дерев пошуку:
- AVL
- Red-Black
- Splay

Проєкт складається з двох окремих модулів:
- `application` — backend (Java + Spring Boot), де виконуються всі алгоритмічні обчислення, кроки операцій і benchmark-метрики.
- `ui` — frontend (Vue + D3.js), який відповідає лише за візуалізацію, анімацію та керування сценаріями програвання.

Ключовий архітектурний принцип:
- алгоритми живуть лише на backend;
- frontend не містить логіки балансування дерева, а лише відображає події/знімки, отримані через REST API.

---

# 📌 Загальна стратегія

Мета: створити веб-систему, яка:
- реалізує AVL, Red-Black та Splay дерева

- проводить експериментальні вимірювання
